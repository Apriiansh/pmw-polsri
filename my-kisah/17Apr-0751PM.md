# Kisah: Perbaikan Alur Seleksi Administrasi

## Apa yang Terjadi?
Admin melaporkan error 404 saat mencoba mengakses fitur Seleksi Administrasi. Hal ini disebabkan oleh inkonsistensi antara definisi rute di `Config/Routes.php` dengan link yang ada di sidebar dan controller.

## Apa yang Telah Dilakukan?
1.  **Standardisasi Rute**: Mengubah rute dari `admin/seleksi-administrasi` menjadi `admin/administrasi/seleksi` agar lebih terstruktur dan sesuai dengan struktur direktori view.
2.  **Pembaruan Sidebar**: Memastikan link "Administrasi" di sidebar mengarah ke URL yang benar.
3.  **Pembersihan View**: Memperbarui semua tombol filter, tombol aksi (Detail, Validasi), dan tombol kembali di dalam view `seleksi.php` dan `seleksi_detail.php`.
4.  **Validasi Controller**: Memastikan fungsionalitas validasi (Approve/Reject/Revision) sudah menggunakan URL baru untuk redirect agar tidak terjadi 404 setelah submit data.
