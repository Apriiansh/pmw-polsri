# Kisah: Refactor Tabel Monitoring Admin ke List Card

## Status: 🟢 Selesai

### Apa yang dilakukan:
1. Menghapus `<table>` lama yang sulit dibaca di mobile pada tab Monitoring Admin.
2. Menggantinya dengan sistem `List Card` yang mobile-first dan responsive.
3. Setiap kartu menampilkan informasi tim (Nama Usaha, Ketua, Kategori) dan grid status pengumpulan laporan.
4. Menambahkan fitur search bar (UI) untuk persiapan filter data tim.
5. Mempertegas tombol pratinjau PDF dengan warna rose standard agar selaras dengan sisi Dosen.

### File yang diubah:
- `app/Views/admin/laporan_pmw.php`

### Catatan:
- Tampilan monitoring sekarang jauh lebih bersih dan premium, memudahkan Admin memantau ratusan tim sekaligus tanpa harus scroll horizontal.
