# Jurnal Modernisasi PMW Polsri - 16 April 2026

Berikut adalah catatan perjalanan pengembangan dan modernisasi sistem PMW Polsri yang telah kita lakukan secara bertahap:

## 1. Fondasi Infrastruktur (Vite & Alpine.js)
*   **Migrasi Modern**: Mengganti penggunaan **Alpine.js** versi CDN yang lambat dengan instalasi berbasis NPM melalui **Vite**.
*   **Build System**: Inisialisasi `package.json` dan `vite.config.js` untuk mengelola aset CSS dan JS secara profesional.
*   **Legacy Cleanup**: Update file `layouts/main.php` agar secara otomatis menggunakan aset yang telah dicompile oleh Vite, meningkatkan kecepatan load halaman secara signifikan.

## 2. Refactoring Database & Workflow
*   **Skema Baru**: Menambahkan kolom-kolom kritikal pada tabel `pmw_proposals` untuk mendukung kategori **Pemula** dan **Berkembang**.
*   **Tracking Status**: Menambahkan sistem status validasi bertingkat (Dosen -> Admin) untuk tahap Seleksi Administrasi dan Pitching Desk.
*   **Data Integrity**: Memastikan data NIM, Nama, Jurusan, dan Prodi tersimpan dengan format yang konsisten di semua tahap penugasan tim.

## 3. Modernisasi Form Pengajuan Mahasiswa
*   **Dynamic UI**: Membangun kembali form proposal menggunakan Alpine.js untuk logika input dinamis (misal: otomatis menyembunyikan/menampilkan field berdasarkan kategori usaha).
*   **Premium Design**: Menerapkan standar desain "Polsri Pay" dengan `card-premium` dan sistem tipografi yang lebih elegan.

## 4. Standarisasi Seleksi Administrasi (Tahap 2)
*   **Pola Validasi Baru**: Memperkenalkan pola **Radio Button Card** (Setujui, Revisi, Tolak) yang lebih intuitif bagi Admin.
*   **Member Biodata Modal**: Menambahkan fitur popup informasi detail mahasiswa/dosen tanpa harus berpindah halaman.

## 5. Optimalisasi Pitching Desk (Tahap 3)
*   **In-App Media Player**: Implementasi pemutar video (YouTube/Drive) langsung di dashboard Dosen dan Admin.
*   **In-App Doc Preview**: Menambahkan preview dokumen PPT/PDF secara *inline* menggunakan iframe dan rute server khusus.
*   **Refactoring List View**: Memperbaiki halaman daftar (Index) dengan ikon akses cepat untuk Video dan PPT demi efisiensi kerja Admin dan Dosen.
*   **Bug Fixing**: Perbaikan error `Undefined array key` dan optimasi query subselect untuk mempercepat loading data dokumen.
